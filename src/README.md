# Flight Booking System

## Project Description
This project is a simple flight booking system written in Java.
It demonstrates basic object-oriented programming concepts such as
classes, inheritance, abstraction, arrays, and method overriding.

## How to Run
1. Open the project in IntelliJ IDEA or any Java IDE.
2. Run the `Main` class.
3. The program output will be displayed in the console.

## Implemented Assignment Requirements
- Created classes: `Flight`, `Passenger`, `Booking`
- Created abstract class `Person`
- Implemented inheritance (`Passenger` extends `Person`)
- Used an abstract method (`getRole`)
- Worked with an array of `Flight` objects
- Implemented search of a flight by flight number
- Filtered flights by available seats
- Sorted flights by available seats
- Overrode `toString()`, `equals()`, and `hashCode()`
- Implemented booking with seat reservation
